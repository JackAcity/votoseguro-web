# Voto Seguro 2026

Simulador educativo de cedula de sufragio para las Elecciones Generales del Peru - 13 de abril de 2026.

## Demo en vivo

https://votoseguro-web.vercel.app

## Stack

- Next.js 16 + TypeScript + Tailwind CSS
- Deploy: Vercel
- CI/CD: GitHub Actions

## Desarrollo local

npm install && npm run dev

## Las 5 columnas de la cedula

1. Formula Presidencial (sin preferencial)
2. Senadores Nacionales (hasta 2 preferenciales)
3. Senadores Regionales (hasta 1 preferencial)
4. Diputados (hasta 2 preferenciales)
5. Parlamento Andino (hasta 2 preferenciales)

## Aviso Legal

Herramienta educativa. La cedula oficial es emitida exclusivamente por la ONPE.
